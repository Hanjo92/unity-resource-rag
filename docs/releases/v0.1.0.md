# v0.1.0

Initial public scaffold for `unity-resource-rag`.

## Highlights

- Added the UPM package `com.hanjo92.unity-resource-rag` for Unity-side execution.
- Added project resource indexing and catalog export for real UI assets.
- Added `apply_ui_blueprint` to build UI from resolved asset-aware blueprints.
- Added the Python sidecar pipeline for planning, retrieval, binding, and verification.
- Added an MCP stdio server wrapper so the sidecar can be called directly as MCP tools.
- Added local heuristic fallback for reference extraction when no API key is available.

## Included In This Release

### Unity UPM Package

- `index_project_resources`
- `ui_asset_catalog`
- `apply_ui_blueprint`
- sample blueprints
- package README and package metadata

### Sidecar Pipeline

- reference image to layout extraction
- layout to blueprint template generation
- vector index build and hybrid retrieval
- blueprint asset binding
- MCP handoff bundle generation
